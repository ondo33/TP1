# Vérification de l'âge et affichage des nombres pairs

## Description
Ce programme Python permet :
- de demander le nom de l’utilisateur
- de saisir l’âge de l’utilisateur
- de vérifier si l’utilisateur est majeur
- si l’utilisateur est mineur, il doit retaper son âge
- d’afficher tous les nombres pairs de 1 à 100

Ce programme utilise les structures de base du langage Python.

---

## Prérequis
- Python 3.x
- Un éditeur de code (Visual Studio Code )

---

##  Utilisation
1. Ouvrir le fichier Python
2. Lancer le programme avec la commande :
```bash
python main.py
